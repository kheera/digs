# Get Digs: Dynamic Interactive Goal Setting

* Set goals
* Track progress
* Document difficulties
* Create billing reports
* Export billing reports to invoice systems

## How to:

* command line: 
  * node timerAppHowAmI.mjs
* UI:
  * cd get-digs-ui
  * npm run start
* Backend:
  * (from project root)
  * node app.js
